# Personal Finance Manager (C)

A simple menu-driven console application built using C programming to manage personal income, expenses, and balance.

---

## Features

* Add Income
* Add Expense
* View Balance
* View Transaction History
* Save Transactions to File
* Load Transactions from File
* Clear Transaction History
* Interactive Menu System

---

## Tech Stack

* C Programming
* Arrays
* File Handling
* Loops (while, for)
* Switch-case
* Basic Input/Output

---

## Future Improvements

* Structures for better data management
* Transaction Categories (Food, Travel, Shopping, etc.)
* Monthly Expense Reports
* Search Transactions

---

## How to Run

```bash
gcc finance_manager.c -o finance_manager
./finance_manager
```
